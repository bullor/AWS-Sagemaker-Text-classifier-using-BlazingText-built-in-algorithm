# AWS-Sagemaker-Text-classifier-using-BlazingText-built-in-algorithm

### Introduction

In this repo  SageMaker BlazingText built-in algorithm is used to predict the sentiment for each customer review. BlazingText is a variant of FastText which is based on word2vec.
